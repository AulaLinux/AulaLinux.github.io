En la carpeta de iconos encontrareis los iconos que disponemos para cada script. 
Se puede poner cualquier otro icono y debéis cambiar en cada script el path del icono donde lo coloquéis. 

Los scprits para que se vean en el botón de aplicaciones o inicio se deben colocar en el path: 
		/local/share/applications
Lo mejor es usar el comando:
		 cp *.desktop /local/share/applications
